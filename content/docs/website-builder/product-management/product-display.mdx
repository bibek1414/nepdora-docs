## üßæ Product Management Options

### 1. **Basic Information**

<Callout type="info" title="Tip">

### 2. **Pricing & Stock**

Control your product pricing and availability.

| Field              | Description                                                              |
| ------------------ | ------------------------------------------------------------------------ |
| **Price**          | The main selling price.                                                  |
| **Compare Price**  | (Optional) Display a discounted price with strikethrough for promotions. |
| **Stock Quantity** | Number of available units.                                               |

---

### 3. **Inventory & Variants**

- Toggle **‚ÄúEnable stock tracking‚Äù** to show availability (e.g., _In Stock_, _Out of Stock_).
- Use **variants** to manage multiple options like:

  - Material (Cotton, Leather)

<Callout type="tip" title="Best Practice">
  Variants automatically generate unique combinations for easy inventory
  tracking.
</Callout>

---

### 4. **Categories**

Organize products by assigning categories and subcategories.

| Field           | Description                                                  |
| --------------- | ------------------------------------------------------------ |
| **Category**    | Choose a primary category (e.g., _Electronics_, _Clothing_). |
| **Subcategory** | Select a nested subcategory (e.g., _Laptops_, _T-Shirts_).   |

Proper categorization helps users filter and find products easily.

---

### 5. **Images**

#### üñºÔ∏è Main Image

- **Click to upload or drag and drop.**
- Maximum file size: **5 MB**
- Supported formats: **JPG, PNG, WebP, GIF**

#### üñºÔ∏è Gallery Images

<Callout type="info" title="Accessibility">

</Callout>

---

### 6. **Product Settings**

Control how your product is displayed on the site.

| Setting                          | Description                                                   |
| -------------------------------- | ------------------------------------------------------------- |

| **Mark as Popular**              | Tag the product as trending or best-selling.                  |
