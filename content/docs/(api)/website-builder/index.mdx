# Website Builder

API for building and customizing your website.

## Overview

The website builder API enables you to:

- Create and manage website pages
- Customize themes and layouts
- Configure domain settings
- Manage site components
- Set up navigation and footers

## Website Components

### Pages

- `GET /api/pages/` - List pages
- `POST /api/pages/` - Create page
- `PUT /api/pages/{id}/` - Update page
- `DELETE /api/pages/{id}/` - Delete page

### Page Components

- Reusable page components
- Component library management
- Component publishing and versioning

### Navigation

- `GET /api/navbar/` - Get navigation settings
- `PUT /api/navbar/{id}/` - Update navigation
- Navigation publishing and management

### Footer

- `GET /api/footer/` - Get footer settings
- `PUT /api/footer/{id}/` - Update footer
- Footer publishing and management

### Themes

- `GET /api/themes/` - List themes
- `POST /api/themes/` - Create theme
- Theme customization and publishing

### Domains

- `GET /api/domains/` - List domains
- `POST /api/domains/` - Add domain
- Domain configuration and SSL

### Services

- Service page management
- Service descriptions and pricing

## Site Management

- Multi-page website support
- Responsive design capabilities
- SEO optimization tools
- Content publishing workflow
