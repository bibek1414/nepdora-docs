# API Documentation

Welcome to the Nepdora API documentation. Our REST API provides comprehensive access to all platform features.

## Getting Started

Before you begin, make sure you have:

- A valid API key
- Understanding of REST principles
- JSON format knowledge

## Authentication

All API requests require authentication. Include your API key in the request headers:

```bash
Authorization: Bearer YOUR_API_KEY
```

## Base URL

```
https://api.nepdora.com
```

## Rate Limits

- **Free tier**: 100 requests per hour
- **Pro tier**: 1,000 requests per hour
- **Enterprise**: Custom limits

## Error Handling

The API uses standard HTTP status codes and returns detailed error messages in JSON format.

## API Sections

Our API is organized into the following sections:

- **[Authentication](/docs/api/authentication)** - User authentication and authorization
- **[User Management](/docs/api/user-management)** - User accounts, teams, and invitations
- **[Content Management](/docs/api/content-management)** - Blogs, portfolios, FAQs, and media
- **[E-commerce](/docs/api/e-commerce)** - Products, orders, categories, and inventory
- **[Website Builder](/docs/api/website-builder)** - Pages, themes, and site configuration
- **[System](/docs/api/system)** - Analytics, integrations, and system utilities
