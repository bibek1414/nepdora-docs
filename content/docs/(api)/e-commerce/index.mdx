# E-commerce

Complete e-commerce API for managing your online store.

## Overview

The e-commerce API provides comprehensive functionality for:

- Product catalog management
- Order processing and fulfillment
- Category and inventory organization
- Customer wishlists and reviews
- Store profile configuration

## Core Features

### Product Management

- `GET /api/product/` - List products
- `POST /api/product/` - Create product
- `PUT /api/product/{id}/` - Update product
- `DELETE /api/product/{id}/` - Delete product

### Order Processing

- `GET /api/order/` - List orders
- `POST /api/order/` - Create order
- Order status tracking and updates
- Customer order history

### Category Management

- Product categories and subcategories
- Hierarchical organization
- Category-specific settings

### Inventory

- Product image management
- Stock tracking
- Bulk upload capabilities

### Customer Features

- Wishlist management
- Product reviews and ratings
- Order tracking

### Store Management

- Store profile configuration
- Multi-store support
- Store-specific settings

## Payment Integration

The API supports integration with various payment providers for seamless checkout experiences.
