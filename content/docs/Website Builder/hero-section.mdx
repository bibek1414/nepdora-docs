## How to Add a Hero Section

1. Go to the **Components** section in the sidebar.
2. Click **Add Component** â†’ **Hero Section**.
3. Select a **Hero Template**:
   - `Image Hero`
   - `Split Hero`
   - `Video Hero`
4. Customize the content:
   - Upload or select a background image
   - Edit headline, subtext, and button text
5. Customize styles:
   - Change text colors, background overlays, and button colors
   - Adjust font sizes, styles, and spacing
6. Click **Save** to apply the hero section to your page.
