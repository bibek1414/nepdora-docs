# Adding a Footer

A **Footer** is displayed at the bottom of your page and typically includes:

- 🌐 Links to important pages (Privacy Policy, Contact, Terms)
- 🎨 Customizable colors, fonts, and layout
- 📱 Social media icons
- ⚡ Optional sections like newsletter signup or copyright notice

---

## Footer Templates

Nepdora offers several footer templates:

- **Simple Footer** – Minimal with essential links
- **Link Footer** – Organizes links into columns for better navigation
- **Social Footer** – Includes social media icons and links

---

## How to Add a Footer

1. Go to the **Components** section in the sidebar.
2. Click **Add Component** → **Footer**.
3. Select a **Footer Template**:
4. Customize the content:
   - Add page links (`Privacy Policy`, `Contact`, `Terms`)
   - Add social icons (`Facebook`, `Twitter`, `Instagram`)
   - Include text or copyright notices
5. Customize styles:
   - Change colors, fonts, and spacing
   - Enable dark/light mode if needed
6. Click **Save** to apply the footer to your page.
